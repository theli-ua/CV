A network limiter may include having a mapping manager that keeps track of which IP addresses are served by which destination hosts or destination service in a computing service environment. There may be a token bucket for each destination host or destination service. Each time a packet is sent to an IP address served by the destination host or destination service, a token may be removed from the bucket and the packet may be forwarded to the designated IP address. If the bucket is empty, the network limiter may drop packet packets destined for any IP address served by the destination host or destination service in order to prevent the destination host or destination service from being overloaded.}
